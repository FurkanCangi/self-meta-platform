Bu dizin, DNA Asistanı kataloğunun dördüncü araştırma paketi grubuna ait kanonik Markdown kaynaklarını saklar.

- `stress-systems.md`: Stres sistemleri
- `sleep-daily-rhythm.md`: Uyku ve günlük ritim

Dosyalar araştırma paketlerinden içerik değiştirilmeden alınmıştır. Burada bulunmak bir iddianın canlı yanıta açıldığı anlamına gelmez. Runtime yalnız kaynak kimliği doğrulanmış, güvenlik sınırı açık ve katalogda ayrıca tanımlanmış alt kümeyi kullanır. Tüm ham kayıtların uzman içerik incelemesi beklemektedir.
